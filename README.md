A análise de agrupamentos são técnicas aplicadas com o objetivo de identificar e agregar objetos, a partir de suas próprias características (Joseph et al., 2009, p. 430).

O presente trabalho consiste na realização de uma análise de agrupamentos para fazer uma análise dos usuários de uma plataforma de streaming, com o objetivo de verificar se há agrupamentos latentes na base de dados e identificá-los caso existam.
